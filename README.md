Shopping Cart Web Application
Overview

Developed a dynamic and responsive Shopping Cart web application that simulates core e-commerce functionalities. The project demonstrates strong frontend development skills, including DOM manipulation, event handling, and real-time state updates.

Key Features
Interactive product listing interface
Add, remove, and update cart items
Real-time price calculation and cart updates
Responsive design for multiple screen sizes
Efficient handling of user interactions
Tech Stack
Frontend: HTML, CSS, JavaScript
Deployment: GitHub Pages
Core Concepts Applied
DOM Manipulation and Event Handling
Dynamic UI Rendering
State Management (Cart Logic)
Responsive Web Design
Project Structure
Shopping_Cart/
│── index.html

│── assets/
How to Run Locally

Clone the repository

git clone https://github.com/codesonali/Shopping_Cart.git
Open index.html in any modern web browser
